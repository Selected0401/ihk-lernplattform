# Emilia's IHK Lernplattform 🐰

Deine persönliche Lernplattform für die IHK Zwischenprüfung Kaufmann für Büromanagement - Teil 1: Informationstechnisches Büromanagement.

## 🚀 Live Demo

**Bald verfügbar unter:** https://alex-ihk-lernplattform.github.io/

## 📱 Features

- ✅ **PWA Installierbar** - Offline lernen wie eine echte App
- ✅ **38+ interaktive Übungen** - Excel, Word, PowerPoint, Outlook
- ✅ **3 AKA Prüfungssimulationen** - Echte Prüfungsbedingungen
- ✅ **Emilia Coach Chat** - Deine persönliche Lern-Coachin
- ✅ **Progress Tracking** - Detaillierte Lernfortschritte
- ✅ **Morning Command Center** - Tägliche Lern-Planung

## 📋 Installation

1. **Browser öffnen:** https://alex-ihk-lernplattform.github.io/
2. **PWA installieren:** 
   - Chrome: "Installieren" Button in Adressleiste
   - Safari: "Zum Home-Bildschirm hinzufügen"
3. **Offline lernen:** Die App funktioniert auch ohne Internet!

## 🎯 Prüfungsvorbereitung

Die Lernplattform ist speziell für die **IHK Zwischenprüfung Teil 1 (Februar 2026)** ausgerichtet:

- **120 Minuten Prüfungszeit**
- **Alle relevanten Office-Anwendungen**
- **DIN 5008 Konformität**
- **AKA Musteraufgaben integration**

## 💡 Lern-Tipps

1. **Täglich üben** - 15 Minuten konsequent sind besser als 2 Stunden alle 2 Wochen
2. **Schwachstellen fokussieren** - Die Analytics zeigen dir wo du üben solltest
3. **Prüfungssimulationen** - Mach regelmäßig die 120-Minuten-Simulationen
4. **Emilia Coach fragen** - Ihre Tipps sind Gold wert!
